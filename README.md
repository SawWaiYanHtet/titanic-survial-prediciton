# Titanic Survival Prediction

## Overview
Predict survival of Titanic passengers using ML.

## Steps
1. Data Cleaning (missing values)
2. EDA (survival by gender)
3. Model Training (Logistic Regression, Random Forest)

## Results
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 81.0% |
| Random Forest | 82.1% |

## Key Finding
Female survival rate (74%) >> Male (19%)
